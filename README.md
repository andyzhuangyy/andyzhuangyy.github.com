## install

修改 gem 源, 参考[ruby-china](https://gems.ruby-china.com/)

    gem sources --add https://gems.ruby-china.com/ --remove https://rubygems.org/

列出 gem 源

    gem sources -l

安装 jekyll

    gem install jekyll

创建博客

    jekyll new myBlog

启动

    jekyll serve
